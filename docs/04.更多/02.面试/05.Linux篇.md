## 1. 常用命令
- su：切换用户
- pwd：查看当前路径
- cd：调整路径
- cp：复制文件
- mv：移动文件
- mkdir：创建目录
- touch：创建文件
- rm：删除目录
- cat：浏览文件
- ls：列出目录文件
- chmod：修改目录权限
- top：查看cpu、内存等信息
- ps：查看进程
- kill：杀掉进程
- netstat：查看端口

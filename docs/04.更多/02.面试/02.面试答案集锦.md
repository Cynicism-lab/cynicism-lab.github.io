1. **Object 九大方法介绍：**

| 方法          | 描述           | 
| ------------- |:-------------:|
| clone         | 返回对象的副本            |
| equals        | 比较两个对象的地址       |
| finalize      | 已过时                    |
| getClass      | 返回此Object的运行时类        |    
| hashCode      | 返回对象的哈希码值|
| toString      | 返回对象的字符串表示形式|
| notify        | 唤醒正在此对象监视器上等待的单个线程|
| notifyAll      | 唤醒等待此对象监视器的所有线程|
| wait          | 导致当前线程等待它被唤醒

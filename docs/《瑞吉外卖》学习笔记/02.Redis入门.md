:::tip Redis简介
Redis（Remote Dictionary Server），即远程字典服务，是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API，属于非关系型数据库
:::
## 1. 数据类型
- 字符串：String（常用）
- 哈希：hash（适合存储对象）
- 列表：list（任务队列）
- 集合：set
- 有序集合：sorted set（排行榜）
